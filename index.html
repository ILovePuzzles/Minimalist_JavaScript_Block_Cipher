MINIMALIST SUBSTITUTION AND TRANSPOSITION CIPHER





CIPHER INSTRUCTIONS


The library cannot contain the same character more than once;
The library must contain at least 2 characters, up to a maximum of 256 characters;
The input message and the two keys can only contain characters from the library;
The two keys must not be longer than the input message;
If your message and the keys are not a multiple of the length of the library, then additionnal characters will be added at the end;
If you choose to complete your keys using pseudorandom characters, then you may leave the input blank, or choose a seed from 0 to 65536.









Define the PRNG and seed values, or leave blank to generate values automatically
